HyperLinkBuilder.jsx
Javascript pour InDesign CC, CS6, CS5.5, CS5
PURPOSE:
Convertir n'importe quel texte URL stylé avec un style de caractère à un lien hypertexte ou lien e-mail.
Création des:
- Hyperliens d'URL
- Hyperliens Email
Ces hyperliens fonctionnent également dans des fichiers PDF interactifs.
Création d'hyperliens
Convertir les adresses web:
Ces paramètres:
Convertir en hyperliens le texte qui est de style de caractère:
URL
^

Avant:http://
+'texte trouvé'+
Apres:
vous permet de convertir du texte stylé avec le style de caractère 'URL'
au liens valide cliquables vers des sites Web externes.
Le texte dans InDesign peut-être de ces types:
aiedv.ch
www.aiedv.ch
http://aiedv.ch
http://www.aiedv.ch

Les hyperliens dans InDesign ressemblera à ceci:
http://www.aiedv.ch
Convertir les adresses de courriel:
Ces paramètres:
Convertir en hyperliens le texte qui est de style de caractère:
email
^

Avant:mailto:
+'texte trouvé'+
Apres:?subject=concerne:
vous permet de convertir du texte stylé avec le style de caractère 'email'
au liens valide cliquables vers des adresses email.
Le sujet est présélectionné avec 'concerne: '.
Le texte dans InDesign peut-être de ce type uniquement:
ai@aiedv.ch

Les hyperliens dans InDesign ressemblera à ceci:
mailto:ai@aiedv.ch?subject=concerne:
Convertir les adresses Web avec des paramètres:
Exemple: création d'article remet en une boutique en ligne comme «ShopWare».
Ces paramètres:
Convertir en hyperliens le texte qui est de style de caractère:
ArticleNum
^

Avant:http://www.mydomain.com/shop/shopware.php/sViewport,checkout/sAction,addArticle/sAdd,A
+'texte trouvé'+
Apres:
vous permet de convertir du texte stylé avec le style de caractère 'URL'
au liens valide cliquables vers des sites Web externes.
Supposons, les parties de texte stylisé sont des numéros d'article comme '123-45678'

Les hyperliens dans InDesign ressemblera à ceci:
http://www.mydomain.com/shop/shopware.php/sViewport,checkout/sAction,addArticle/sAdd,A123-45678
Cela va pousser l'article 123-45678 directement dans le panier.
Rangement liens existants
Hyperliens déjà définis dans InDesign peuvent être «nettoyées».
- supprimer orphelins liens cibles d'URL
- supprimer tous les liens hypertextes actuellement définies

Les paramètres dans le panneau 'Ranger hyperliens' sont également efficaces avant de créer les nouveaux liens.
En cliquant sur​le bouton "Ranger Hyperliens» effectue un «nettoyage» seulement.
Stockage et du rechargement des paramètres
Les paramètres actuellement affichés peuvent être stockés sur le disque pour une utilisation ultérieure.
Le fichier de paramètres par défaut 'settings.set' est toujours chargé avant d'afficher la boîte de dialogue principale.
Si vous voulez toujours les mêmes paramètres au démarrage, mémoriser vos paramètres préférés comme 'settings.set'.

Appuyer sur le bouton 'Enregistrer paramètres':
Donnez les paramètres un nom auto-description lorsque vous êtes demandé le nom des paramètres.

En appuyant sur le bouton 'Charger paramètres':
Les paramètres précédemment enregistrés peuvent être rechargés et exécutés.

Les paramètres sont stockés dans un dossier nommé «HyperLinkBuilderSupport» dans le dossier de départ de l'utilisateur sur le disque de démarrage.
