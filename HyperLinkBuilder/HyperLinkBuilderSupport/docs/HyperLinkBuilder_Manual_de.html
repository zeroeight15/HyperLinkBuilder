HyperLinkBuilder.jsx
Javascript für InDesign CC, CS6, CS5.5, CS5
ZWECK:
Konvertiert beliebigen Text der mit einer bestimmten Zeichenstilvorlage markiert ist zu klickbaren Internet Links oder zu Email-Links.
Automatische Kreation von:
- URL Hyperlinks
- Email Hyperlinks
Diese Hyperlinks funktionieren auch in interaktiven PDF Dateien.
Kreieren von Hyperlinks
Internet-Adressen konvertieren:
Diese Einstellungen:
Text zu Links konvertieren der markiert ist mit:
URL
^

Vor:http://
+'gefundener Text'+
Nach:
konvertieren Text, wenn markiert mit der Zeichenstilvorlage 'URL',
zu klickbaren Links zu externen Webseiten.
Der gestylte Text kann in den folgenden Formen vorkommen:
aiedv.ch
www.aiedv.ch
http://aiedv.ch
http://www.aiedv.ch

Der Ziellink in InDesign wird so aussehen:
http://www.aiedv.ch
Convert Email Addresses:
Diese Einstellungen:
Text zu Links konvertieren der markiert ist mit:
email
^

Vor:mailto:
+'gefundener Text'+
Nach:?subject=Betrefftext:
konvertieren Text, wenn markiert mit der Zeichenstilvorlage 'email',
zu klickbaren Email-Links die das Emailprogramm des lokalen Computers starten.
Die Betreffzeile wird ausgefüllt mit 'Betrefftext: '
Der gestylte Text kann in dieser Form vorkommen:
ai@aiedv.ch

Der Ziellink in InDesign wird so aussehen:
mailto:ai@aiedv.ch?subject=Betrefftext:
Internet-Adressen mit Parameter:
Beispiel: Artikel-Aufrufe in einen Online-Shop wie 'ShopWare'.
Diese Einstellungen:
Text zu Links konvertieren der markiert ist mit:
ArtikelNum
^

Vor:http://www.mydomain.com/shop/shopware.php/sViewport,checkout/sAction,addArticle/sAdd,A
+'gefundener Text'+
Nach:
konvertieren Text, wenn markiert mit der Zeichenstilvorlage 'ArtikelNum',
zu klickbaren Links zu externen Webseiten.
Nehmen wir an, dass der gestylte Text Artikelnummern sind wie z.B. '123-45678'

Der Ziellink in InDesign wird so aussehen:
http://www.mydomain.com/shop/shopware.php/sViewport,checkout/sAction,addArticle/sAdd,A123-45678
Dies legt den Artikel 123-45678 direkt in den Warenkorb.
Existierende Hyperlinks aufräumen
Hyperlinks die bereits in InDesign definiert sind können 'bereinigt' werden.
- Entfernen von verwaisten URL Ziellinks
- Entfernen ALLER momentan definierten Hyperlinks

Die Einstellungen im Bereich 'Hyperlinks aufräumen' sind auch effektiv BEVOR die neuen Hyperlinks kreiert werden.
Mit einem Klick auf den 'Hyperlinks aufräumen' Button wird nur die 'Bereinigung' ausgeführt.
Speichern und Laden von Einstellungen
Die aktuell angezeigten Einstellungen können auf der Disk für den späteren Gebrauch gespeichert werden.
Die Standard-Einstellungen in der Datei 'settings.set' werden immer geladen bevor der Hauptdialog angezeigt wird.
Falls Sie immer mit einer bestimmten Einstellung starten möchten, speichern Sie diese Einstellungen mit Namen 'settings.set'.

Der 'Einstellungen speichern' Button:
Geben sie der Einstellungsdatei einen Namen der die Einstellungen gut beschreibt.

Der 'Einstellungen laden' Button:
Einmal gespeicherte Einstellungen können zur erneuten Ausführung wieder geladen werden.

Einstellungen werden im Ordner 'HyperLinkBuilderSupport' im Benutzerverzeichnis auf der Startdisk gespeichert.
